# CUU-Student-Record-System
University automated system integrating all key academic and administrative functions
University Student Records Management System (CUU System)
A role-based web prototype developed for Cavendish University Uganda to manage academic records, financial clearance, and administrative workflows.
Core Functionality
Role-Based Access: Distinct interfaces for Students, Lecturers, and Administrators.
Financial Gate Logic: Dynamically restricts access to academic results and exam permits based on real-time fee balances.
Academic Management: Features for course enrollment, automated GPA calculation (5.0 scale), and retake applications.
Semester Roadmap: Sidebar navigation for tracking key academic dates and deadlines.
Technical Stack
Frontend: HTML5, Tailwind CSS, Lucide Icons.
Backend: Node.js, Express.js.
Database: MySQL.
Auth: JWT (JSON Web Tokens).
Installation and Setup
Clone Repository: git clone https://github.com/YourUsername/CUU-Records-System.git
Database: Import backend/schema.sql into MySQL.
Backend: Run npm install and node server.js in the backend folder.
Frontend: Open frontend/index.html in a web browser.
